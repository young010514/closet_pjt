# Closet PJT - 사업자 기능 PRD (제품 요구사항 정의서)

> 작성 대상: 기획자, PM, 디자이너, 비개발 이해관계자
> 작성일: 2026년 6월

---

## 1. 배경 및 목적

### 서비스 소개
Closet PJT는 패션을 주제로 한 커뮤니티 플랫폼입니다. 일반 사용자는 패션 정보를 공유하고 소통하며, 체험단에 신청할 수 있습니다.

### 이번 기능 추가 배경
현재 서비스는 일반 사용자 중심으로 운영되고 있습니다. 이번 업데이트를 통해 **사업자 회원 전용 관리 페이지**를 신설하여 사업자가 가게를 홍보하고 체험단을 운영할 수 있는 환경을 제공합니다.

### 목적
- 사업자 회원이 직접 가게를 홍보하고 고객과 소통할 수 있는 전용 공간 제공
- 체험단 모집과 신청자 관리를 한 곳에서 처리할 수 있도록 운영 효율화
- 일반 사용자 커뮤니티와 자연스럽게 연동되는 구조로 홍보 효과 극대화

---

## 2. 사용자 유형

### 일반 회원
- 커뮤니티에서 패션 정보를 공유하고 소통
- 체험단 공고를 보고 신청 가능
- **사업자 전용 페이지에는 접근 불가**

### 사업자 회원
- 일반 커뮤니티 이용 가능 (일반 회원과 동일)
- **추가로 사업자 전용 관리 페이지 이용 가능**
- 로그인 시 사업자 전용 대시보드로 자동 이동

---

## 3. 주요 기능

사업자 전용 페이지는 3가지 핵심 기능으로 구성됩니다.

---

### 3.1 대시보드

사업자가 로그인하면 가장 먼저 보이는 화면입니다. 내 가게의 운영 현황을 한눈에 확인할 수 있습니다.

**제공 정보**
- 내 가게 기본 정보 요약 (가게명, 주소 등)
- 작성한 가게 홍보 게시글 수
- 체험단 현황 (모집 중 / 모집 완료 / 종료)
- 현재 모집 중인 체험단의 신청자 수
- 최근 작성한 게시글 목록 (최대 5개)
- 가게 관리 및 체험단 관리 페이지 바로가기

**사용 시나리오**
> 사업자 A씨가 아침에 로그인합니다. 대시보드에서 어제 올린 체험단에 신청자가 3명 늘었다는 것을 확인하고, 바로 체험단 관리 페이지로 이동해 신청자 정보를 확인합니다.

---

### 3.2 가게 관리

사업자가 자신의 가게와 상품을 소개하는 게시글을 작성하는 공간입니다. 여기서 작성한 게시글은 **일반 사용자 커뮤니티의 '우리 동네 가게' 게시판에 자동으로 노출**됩니다.

**게시글 종류**

| 종류 | 활용 예시 |
|------|----------|
| 신제품 홍보 | "이번 시즌 신상 린넨 셔츠가 입고됐습니다!" |
| 기본 제품 홍보 | "베스트셀러 데님 재입고 안내" |
| 가게 홍보 | "오픈 5주년 기념 할인 이벤트" |
| 오는 길 & 운영 시간 안내 | "주차 안내 및 영업시간 변경 공지" |

**할 수 있는 것**
- 게시글 작성 (텍스트, 사진, 영상 업로드 가능)
- 게시글 수정
- 게시글 삭제

**커뮤니티 연동 구조**

```
사업자가 가게 관리에서 게시글 작성
            ↓
일반 사용자 커뮤니티 '우리 동네 가게' 게시판에 자동 노출
            ↓
일반 사용자가 게시글 열람, 좋아요, 댓글 가능
```

**사용 시나리오**
> 사업자 B씨가 신제품을 입고받았습니다. 가게 관리 페이지에서 신제품 홍보 게시글을 작성하고 사진을 업로드합니다. 작성 즉시 일반 커뮤니티에도 노출되어 일반 사용자들이 바로 확인할 수 있습니다.

**주의 사항**
일반 사용자가 '우리 동네 가게' 게시판에서 특정 가게를 조회할 때, 해당 가게의 홍보 게시글이 함께 표시됩니다. 이때 게시글과 사업자 정보는 사업장 주소를 기준으로 연결됩니다. 사업자 회원가입 시 등록한 사업장 주소와 게시글에 입력된 가게 위치 주소가 일치하는 경우 같은 가게의 게시글로 간주합니다.

---

### 3.3 체험단 관리

사업자가 체험단 모집 공고를 올리고, 신청자를 관리하는 공간입니다. 작성한 체험단 모집 글은 **일반 사용자 커뮤니티의 "체험단 모집 게시판"에 자동으로 노출**됩니다.


**할 수 있는 것**
- 체험단 모집 공고 작성 (모집 기간, 체험 기간, 제품 설명, 유의사항 등 입력)
- 모집 공고 수정 및 삭제
- 내가 등록한 체험단 목록 및 모집 상태 확인
- 신청자 목록 확인 (이름, 연락처, SNS 계정, 지원 동기)

**체험단 상태**

| 상태 | 의미 |
|------|------|
| 모집 중 | 현재 신청을 받고 있는 상태 |
| 모집 완료 | 모집 기간이 종료된 상태 |
| 종료 | 체험 기간까지 모두 끝난 상태 |

**커뮤니티 연동 구조**

```
사업자가 체험단 관리에서 모집 공고 작성
            ↓
일반 사용자 커뮤니티 '체험단' 게시판에 자동 노출
            ↓
일반 사용자(normal)가 신청서 작성 및 제출
            ↓
사업자가 체험단 관리 페이지에서 신청자 목록 확인
```

**사용 시나리오**
> 사업자 C씨가 신제품 체험단을 모집합니다. 체험단 관리 페이지에서 모집 기간, 제품 설명, 유의사항을 입력하고 공고를 등록합니다. 일반 사용자들이 커뮤니티에서 공고를 보고 신청하면, C씨는 신청자 목록을 확인하고 체험단을 선발합니다.

---

## 4. 제약 사항 및 예외 처리

| 상황 | 처리 방식 |
|------|----------|
| 일반 회원이 사업자 페이지 접근 시도 | 접근 차단, 일반 커뮤니티로 리다이렉트 |
| 로그인하지 않은 사용자가 사업자 페이지 접근 시도 | 로그인 페이지로 이동 |
| 사업자가 체험단 신청 시도 | 신청 불가 안내 메시지 표시 |
| 모집 기간이 지난 체험단에 신청 시도 | 신청 불가 안내 메시지 표시 |
| 이미 신청한 체험단에 재신청 시도 | 중복 신청 불가 안내 메시지 표시 |

## 5. 성공 지표 (예시)

| 지표 | 목표 |
|------|------|
| 사업자 회원 가입 수 | 서비스 오픈 1개월 내 사업자 회원 확보 |
| 가게 관리 게시글 작성 수 | 사업자 1인당 월평균 4건 이상 |
| 체험단 모집 공고 등록 수 | 사업자 1인당 월평균 1건 이상 |
| 체험단 신청 전환율 | 공고 열람자 중 10% 이상 신청 |
